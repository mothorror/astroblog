# BlackWhite 主题语法指南

本文档展示了 BlackWhite 主题支持的所有 Markdown 和 MDX 语法。

## 1. 标签高亮

支持 6 种颜色的标签高亮，用于强调重要内容。

### 可用颜色

[!labelblue]@蓝色高亮@ - 用于一般提示

[!labelorange]@橙色高亮@ - 用于警告提醒

[!labelgrey]@灰色高亮@ - 用于次要信息

[!labelpink]@粉色高亮@ - 用于特别标注

[!labelyellow]@黄色高亮@ - 用于注意事项

[!labelgreen]@绿色高亮@ - 用于成功状态

### 语法示例

```markdown
[!labelblue]@这是蓝色标签@
[!labelorange]@这是橙色标签@
[!labelgrey]@这是灰色标签@
[!labelpink]@这是粉色标签@
[!labelyellow]@这是黄色标签@
[!labelgreen]@这是绿色标签@
```

## 2. 容器区块

用于组织和突出显示特定类型的内容。

### Info 信息框

[!info]@这是一个信息提示框，用于提供补充说明和额外信息。@

### Note 笔记框

[!note]@这是一个笔记框，用于记录重要的要点和备忘内容。@

### Warning 警告框

[!warning]@这是一个警告框，用于提醒用户注意潜在问题。@

### Danger 危险框

[!danger]@这是一个危险框，用于警示严重错误或危险操作。@

### Tip 提示框

[!tip]@这是一个提示框，用于分享有用的技巧和建议。@

### Success 成功框

[!success]@这是一个成功框，用于显示操作成功或积极结果。@

### 语法示例

```markdown
[!info]@信息内容@

[!note]@笔记内容@

[!warning]@警告内容@

[!danger]@危险内容@

[!tip]@提示内容@

[!success]@成功内容@
```

## 3. 自定义区块

支持带标题的自定义内容区块。

### 示例区块

[!div]@标题@自定义区块内容，可以包含任意文本和格式。

### 语法示例

```markdown
[!div]@区块标题@区块内容
```
