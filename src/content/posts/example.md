# 分类和标签使用指南

这是如何使用分类和标签功能的示例文章。

## 文章 Frontmatter

```yaml
---
title: "文章标题"
pubDate: 2026-05-09T16:00:00.000Z
updatedDate: 2026-05-09T16:00:00.000Z  # 可选
description: "文章描述"
category: "分类名称"  # 单个分类
tags: ["标签1", "标签2", "标签3"]  # 多个标签
backgroundImage: "https://example.com/image.jpg"  # 可选
---
```

## 使用说明

### 分类

- **字段**: `category`
- **类型**: 单个分类（字符串）
- **示例**: `category: "技术教程"`
- **用途**: 将文章归类到不同的主题下

### 标签

- **字段**: `tags`
- **类型**: 标签数组
- **示例**: `tags: ["Astro", "前端开发", "静态博客"]`
- **用途**: 为文章添加多个关键词标签

## 最佳实践

1. **分类**: 使用宽泛的主题，如"技术教程"、"生活随笔"、"项目分享"
2. **标签**: 使用具体的关键词，如"Astro"、"React"、"CSS技巧"

## 效果

- 文章会显示在对应的分类页面
- 标签页面会列出所有包含该标签的文章
- 首页和文章详情页都会显示分类和标签
